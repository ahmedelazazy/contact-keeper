# Contact Keeper

Following along [Udemy Course: React Front To Back](https://www.udemy.com/course/modern-react-front-to-back/) by [Brad Traversy](https://github.com/bradtraversy)

A contacts manager application build by:

- React
- Express
- Mongo DB
